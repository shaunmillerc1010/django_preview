Shaun Miller Django Demo

Here is a web application run with the Django framework displaying the cryptocurrencies Bitcoin and Ethereum. Visuals are made in Plotly and Pandas Python Libraries. Website template provided by HTML5 UP
and released for free under the Creative Commons.

Candlestick charts display the open, high, low, and closing price of a currency in a particular time interval (5 minutes here). Charts were last updated December 15th at 5:00 pm (US Central Time - Chicago).
